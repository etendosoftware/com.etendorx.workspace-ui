# Plan — Documento de Testing Manual: Matriz de Procesos Legacy

## Contexto

El usuario solicita un documento con TODOS los procesos legacy del sistema para testearlos
manualmente. La implementacion de LegacyProcessResolver ya esta completa; este documento
es la herramienta de validacion.

**Fuente de datos:** Query SQL sobre la DB `etendodev` (localhost:5432), tablas
`AD_Field`, `AD_Tab`, `AD_Window`, `AD_Column`, `AD_Process`.

**Filtro aplicado:** Solo campos con `AD_Column.AD_Reference_ID=28` (Button) y
`em_obuiapp_process_id IS NULL` (excluye procesos modernos OBUIAPP).
